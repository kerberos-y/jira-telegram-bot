# Jira Telegram Bot

Бот надсилає сповіщення про зміни в Jira в Telegram-групу, дозволяє зв'язати Jira-акаунт та формувати звіт по задачах in progress.

## Встановлення

1. Клонуйте репозиторій
2. Встановіть залежності: `npm install`
3. Скопіюйте `.env.example` -> `.env` та заповніть всі змінні
4. Запустіть міграції БД автоматично при старті

## Запуск

- Локально: `npm start` (потрібен публічний HTTPS-тунель, наприклад ngrok)
- На Railway: автоматичний деплой з Git

## Команди бота

- `/start` – реєстрація, привітання, клавіатура
- `/connect_jira your@email.com` – прив'язати Jira-акаунт
- `/report` – показати звіт
- Кнопка `📊 Звіт` – те саме

## Налаштування Jira

1. Виконайте `npm run setup:jira` – створить проект (якщо відсутній)
2. Додайте вручну статуси: *Todo, In Progress, Ready for QA, Done* в налаштуваннях проекту
3. Виконайте `npm run setup:webhook` – створить webhook на ваш сервер

## Webhook endpoints

- `POST /telegram/<BOT_TOKEN>` – від Telegram
- `POST /webhook/jira` – від Jira
